# Aorus-B550M-Elite-Hackintosh
 
Prebuilt opencore EFI for macos

# Desktop specs
- CPU: AMD Ryzen 5 5600G
- GPU: AMD Radeon RX 6600
- Wifi/Bluetooth: Not installed on this build
- Audio codec: ALC887

- change SMBIOS to a valid SMBIOS
